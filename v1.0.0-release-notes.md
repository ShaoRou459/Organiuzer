# Organiuzer v1.0.0 (Initial Release)

Welcome to the first official release of **Organiuzer**! 

Organiuzer is an AI-powered desktop folder organizer designed to help you declutter, categorize, and manage your files safely and intelligently.

## Key Features

* **🤖 AI-Powered Categorization**
  Automatically analyze and sort your files into logical categories using AI, saving you hours of manual organization.

* **🔍 Smart Deduplication**
  Quickly scan your directories, identify exact duplicate files, and choose which ones to keep or remove to free up valuable storage space.

* **🛡️ Safe Executions & History**
  Never lose a file by mistake. Organiuzer builds a clear execution plan for you to review *before* any files are actually moved, while keeping a detailed history of past actions.

* **📂 Interactive File Management**
  An integrated, drag-and-drop file explorer that allows you to manage your workspace seamlessly. 

* **📊 Disk & Space Analytics**
  Visualize what is taking up space on your drive with built-in interactive graphics and charts.

* **💻 Complete Windows Support**
  Available as both standard installers and portable executables, fully optimized for both Windows AMD64 and ARM64 systems.

## Installation
Simply download the appropriate `.exe` file for your system from the assets below. If you prefer not to permanently install the app, you can use the `portable` versions to run it directly from a flash drive or arbitrary folder!
